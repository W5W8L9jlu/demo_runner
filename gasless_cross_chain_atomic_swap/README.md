# Gasless Cross-Chain Atomic Swap

This repository is a history-preserved extraction of `src/demos/gasless-cross-chain-atomic-swap`
from the upstream `reactive-smart-contract-demos` monorepo.

## Layout

- Source contracts and the original demo guide live in `src/demos/gasless-cross-chain-atomic-swap/`
- Minimal build dependencies live in `lib/openzeppelin-contracts/contracts/` and `lib/reactive-lib/src/`
- Foundry is configured at the repository root via `foundry.toml`

## Build

```bash
forge build
```
